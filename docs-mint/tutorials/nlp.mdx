## Setup

```python
import os
from deepfix_sdk import DeepFixClient

os.environ["DEEPFIX_API_KEY"] = "..."  # Get your API key from DeepCoach

client = DeepFixClient(
    api_url="https://deepfix.delcaux.com",
    timeout=120,
)
```

## Sentence Classification

### Load Dataset

```python
from deepfix_sdk.data.datasets import NLPDataset
from deepfix_sdk.zoo.datasets import load_tweet_emotion_classification

train_data, test_data = load_tweet_emotion_classification(
    as_train_test=True,
    include_embeddings=True,
)

dataset_name = "tweet_emotion_classification"

train_dataset = NLPDataset(
    dataset_name=dataset_name,
    dataset=train_data,
)

val_dataset = NLPDataset(
    dataset_name=dataset_name,
    dataset=test_data,
)
```

### Run Analysis

```python
result = client.get_diagnosis(
    train_data=train_dataset,
    test_data=val_dataset,
    language="english",
)

print(result.to_text())
```

The analysis will surface:

- Label distribution drift between train and test.
- Text outliers (e.g. toxicity).
- Unknown token issues.
- Embedding drift between splits.

## Interpreting Findings

Typical issues for NLP datasets include:

- **Label drift**: different label distributions in train vs test.
- **Outliers**: extreme values in text toxicity or length.
- **Unknown tokens**: gaps in tokenizer vocabulary.
- **Embedding drift**: domain shift between data splits.

DeepFix presents:

- A high-level summary section.
- High/medium/low severity issue tables.
- Recommendations and rationale.
